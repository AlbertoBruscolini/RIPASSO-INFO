## Come testare i diagrammi

1. Apri il file `.mmd` con un editor di testo
2. Copia tutto il contenuto
3. Vai su [mermaid.live](https://mermaid.live)
4. Incolla nel pannello sinistro
5. Il diagramma appare a destra

Oppure usa GitHub: rinomina il file in `.md` racchiudendo il codice in:
\`\`\`mermaid
... contenuto ...
\`\`\`
